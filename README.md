# OOP Movie Library

A movie search application that fetches real movie data from the OMDb API and displays movie details.

## Features

- Search for a movie by title.
- Fetches data from OMDb API
- Press Enter or click the search button to fetch results.
- Displays poster, year, genre, IMDB rating and plot in a card.
- Light/dark toggle for the UI.

## How to Use

1. Enter a movie name in the search input.
2. Press Enter or click the `Search` button.
3. The app will show "Loading..." then show the movie card or an error message.

## Technologies Used

- HTML
- CSS – Styling & Layout
- JavaScript
- OMDb API – Movie database
